## Installation

```bash
npm install nexus-bus
```

Local development:

```bash
cd sdk/javascript
npm install
npm run build
```

## Quickstart

```typescript
import { NexusAgent } from "nexus-bus";

const agent = new NexusAgent({
  name: "writer-agent",
  agentType: "writer",
  subscribeTopic: "nexus.analysis",
  nexusUrl: "http://localhost:8000",
  apiKey: "nxs_live_sk_your_key_here",
});

agent.onMessage(async (message) => {
  const top5 = await agent.context.get(message.correlation_id, "top5_stories");
  await agent.publish("nexus.writing", { status: "done" }, message.correlation_id);
});

await agent.start();
```

## NexusAgent

### Constructor options

```typescript
new NexusAgent({
  name: string;
  agentType: string;
  subscribeTopic: string;
  nexusUrl: string;
  apiKey: string;
  heartbeatIntervalMs?: number;  // default 10000
  pollIntervalMs?: number;       // default 3000
})
```

### Methods

| Method | Returns | Description |
| --- | --- | --- |
| `register()` | `Promise<void>` | Register with Nexus |
| `deregister()` | `Promise<void>` | Deregister from Nexus |
| `publish(topic, payload, correlationId)` | `Promise<NexusMessage>` | Publish a message |
| `onMessage(callback)` | `void` | Register async message handler |
| `start({ skipExisting? })` | `Promise<void>` | Register, heartbeat, consume |
| `stop()` | `Promise<void>` | Stop and deregister |

### `agent.context` — ContextClient

| Method | Returns | Description |
| --- | --- | --- |
| `get(correlationId, key)` | `Promise<T \| null>` | Read one key |
| `set(correlationId, key, value, ttlHours?)` | `Promise<{ status, key }>` | Write one key |
| `getAll(correlationId)` | `Promise<Record<string, unknown>>` | All keys for a run |

## NexusMessage

```typescript
type NexusMessage = {
  id: string;
  topic: string;
  correlation_id: string;
  sender_agent: string;
  payload: Record<string, unknown>;
  timestamp?: string;
};
```

## Exceptions

| Error | When thrown |
| --- | --- |
| `NexusAuthError` | Invalid API key |
| `NexusConnectionError` | Cannot reach Nexus API |
| `NexusPublishError` | Publish rejected |
| `NexusConsumeError` | Message polling failed |

```typescript
import { NexusAuthError, NexusPublishError } from "nexus-bus";

try {
  await agent.register();
} catch (error) {
  if (error instanceof NexusAuthError) {
    console.error("Check your API key");
  }
}
```
