---
title: Agents
description: Agent registration, heartbeats, listing, and deregistration.
---

Base URL: `http://localhost:8000/api/v1/agents`

## POST `/agents/register`

Register a new agent.

**Auth:** API key

```json
{
  "name": "writer-agent",
  "agent_type": "writer",
  "subscribe_topics": ["nexus.analysis"]
}
```

**Response:** `200` — Agent object. `409` if already registered.

## DELETE `/agents/{name}`

Deregister an agent.

**Auth:** Admin JWT

**Response:** `200` — `{ "message": "Agent {name} deregistered" }`

## GET `/agents`

List all registered agents.

**Auth:** JWT

**Query:** `status` — optional filter (`active`, `idle`, `dead`)

**Response:** `200` — Array of agents with enriched `messages_processed` and `error_count`.

## GET `/agents/{name}`

Get one agent's details.

**Auth:** JWT

## GET `/agents/{name}/errors`

List dead-letter queue entries for an agent.

**Auth:** JWT

## POST `/agents/{name}/heartbeat`

Send a heartbeat. Called automatically by the SDK every 10 seconds.

**Auth:** API key

**Response:** `200` — `{ "status": "ok", "agent": "...", "timestamp": "..." }`
