---
title: Dead Letter Queue
description: List, retry, and discard failed messages.
---

Base URL: `http://localhost:8000/api/v1/deadletter`

## GET `/deadletter`

List all dead letter records, newest first.

**Auth:** JWT

**Response:** `200` — Array of DLQ records with `original_message`, `error`, `failed_at`, `retry_count`.

## GET `/deadletter/{message_id}`

Get one DLQ record.

**Auth:** JWT

## POST `/deadletter/{message_id}/retry`

Re-publish the original message to its topic.

**Auth:** Admin JWT

**Response:** `200` — `{ "message": "...", "new_message_id": "..." }`

## DELETE `/deadletter/{message_id}`

Permanently discard a dead letter message.

**Auth:** Admin JWT

**Response:** `200`
