---
title: Messages
description: Publish and retrieve messages on the Nexus bus.
---

Base URL: `http://localhost:8000/api/v1/messages`

## POST `/messages/publish`

Publish a message to a topic.

**Auth:** API key or JWT

```json
{
  "topic": "nexus.research",
  "correlation_id": "run_abc123",
  "sender_agent": "scout-agent",
  "payload": { "status": "research_done", "count": 10 }
}
```

**Response:** `200` — Full `NexusMessage` object.

**Side effects:** Message stored in Redis (7-day TTL), pipeline tracker updated, `message.published` WebSocket event broadcast. Auto-completes pipeline when publishing to `nexus.delivery` with `delivery_complete` status.

## GET `/messages`

List recent messages.

**Auth:** API key or JWT

**Query parameters:**

| Param | Description |
| --- | --- |
| `topic` | Filter by topic |
| `agent` | Filter by sender agent |
| `limit` | Max results (default `50`) |

**Response:** `200` — Array of messages, newest first.

## GET `/messages/{message_id}`

Get one message by ID.

**Auth:** API key or JWT

**Response:** `200` — Message object. `404` if not found or expired.
