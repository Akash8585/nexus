---
title: Context
description: Read and write shared pipeline context in Redis.
---

Base URL: `http://localhost:8000/api/v1/context`

## GET `/context/{correlation_id}`

Get all context keys for a pipeline run.

**Auth:** API key or JWT

**Response:** `200` — Object mapping key names to values.

## GET `/context/{correlation_id}/{key}`

Get one context value.

**Auth:** API key or JWT

**Response:** `200` — Value (JSON). `404` if key not found.

## PUT `/context/{correlation_id}/{key}`

Write a context value.

**Auth:** API key

```json
{
  "value": { "stories": [] },
  "ttl_hours": 24
}
```

**Response:** `200` — `{ "status": "ok", "key": "..." }`

## DELETE `/context/{correlation_id}`

Delete all context for a run.

**Auth:** Admin JWT

## DELETE `/context/flush-all`

Delete **all** context data. Destructive — admin only, used for dev reset.

**Auth:** Admin JWT

**Response:** `200` — `{ "message": "...", "deleted": <count> }`
