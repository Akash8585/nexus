## What is an agent?

An **agent** is any program that connects to Nexus to send or receive messages. In the demo pipeline:

- **Scout** — researches headlines
- **Analyst** — ranks stories
- **Writer** — drafts the briefing
- **Deliverer** — saves the output file

Each agent has a unique `name`, an `agent_type`, and one or more `subscribe_topics`.

## Registration

When an agent starts, it calls `POST /api/v1/agents/register` with:

```json
{
  "name": "writer-agent",
  "agent_type": "writer",
  "subscribe_topics": ["nexus.analysis"]
}
```

Registration requires a valid **API key**. Nexus adds the agent to the registry and broadcasts an `agent.registered` WebSocket event to the dashboard.

The SDK handles registration automatically when you call `agent.start()`.

## Heartbeats

Alive agents send a heartbeat every **10 seconds** to:

```
POST /api/v1/agents/{name}/heartbeat
```

If Nexus receives no heartbeat for **30 seconds**, the agent is marked **dead** in the registry and the dashboard updates in real time.

## Agent status

| Status | Meaning |
| --- | --- |
| `active` | Heartbeat received recently |
| `idle` | Registered but quiet |
| `dead` | No heartbeat for 30+ seconds |

View all agents in the dashboard under **Agents**, or via `GET /api/v1/agents`.

## Deregistration

Admins can remove an agent with `DELETE /api/v1/agents/{name}`. The SDK calls deregister on `stop()` for clean shutdown.

## SDK usage

<CodeGroup>

```python Python
from nexus_bus import NexusAgent

agent = NexusAgent(
    name="analyst-agent",
    agent_type="analyst",
    subscribe_topic="nexus.research",
    nexus_url="http://localhost:8000",
    api_key="nxs_live_sk_...",
)

agent.register()   # manual
agent.start()      # register + heartbeat + consume
agent.stop()       # stop + deregister
```

```typescript JavaScript
import { NexusAgent } from "nexus-bus";

const agent = new NexusAgent({ /* ... */ });
await agent.register();
await agent.start();
await agent.stop();
```

</CodeGroup>

See [Python SDK](/sdk/python) and [JavaScript SDK](/sdk/javascript) for full API details.
